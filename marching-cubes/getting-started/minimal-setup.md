# Minimal Setup

Locate the Prefabs Folder and drag the "MarchingCubesManager" and the "SampleVolume" Prefabs into the scene.&#x20;

<figure><img src="../../.gitbook/assets/MarchingCubePrefab.png" alt=""><figcaption></figcaption></figure>

<p align="center"></p>

\
Locate the MainCamera in the scene and add the component "BasicWasdController" to it.

<div align="center"><figure><img src="../../.gitbook/assets/wasd.png" alt=""><figcaption></figcaption></figure></div>



Optional - RightClick > UI > Image, and set the transform as shown below. This places a simple square crosshair in the middle of your screen to visualize where digging and terraforming operations are aimed.&#x20;

<figure><img src="../../.gitbook/assets/CrossHair Setup.png" alt=""><figcaption></figcaption></figure>

\
\
Now you're all set to enter play mode and fly around the scene modifying the SampleVolume with keybinds 1 , 2,  and 3. See what you can tweak in this minimal setup, and consult the documentation for the full package capabilities.&#x20;
