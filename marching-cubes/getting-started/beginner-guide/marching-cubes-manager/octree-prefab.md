# Octree Prefab

One of the Marching Cube Manager's functions is object pooling; it's OctreePrefab field is the object it pools. This is the object with the mesh generated by the Marching Cube Algorithm. This prefab should have a MeshFilter, MeshRenderer with material assigned, and MeshCollider. It should also be assigned to the layer and tagged with the tag you want as default.&#x20;
