# Marching Cubes Manager

The Marching Cubes Manager is the Singleton that manages materials, object pooling and several other low-level processes.\
\
As a user you only need to know to include it in the scene, and to assign it's fields for [leaf-prefab.md](leaf-prefab.md "mention") and [voxel-material-database.md](voxel-material-database.md "mention").&#x20;
